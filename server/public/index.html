What is this?

This is a web server that provides:
  • An OGraf Renderer (a web page) to be loaded in a HTML renderer (such as CasparCG, OBS, Vmix, etc),
  • An API where OGraf Graphics can be uploaded and managed.
  • An API that can be used by a Controller to control the OGraf graphics.
  • A simple Controller web page to control OGraf graphics.

How to use

  1. Upload OGraf Graphics by zipping the OGraf graphic folders.
    You can find example OGraf Graphics here (download zip file)
  2. Open an OGraf Renderer in your HTML renderer (such as CasparCG, OBS, Vmix, etc).
    Use the URL: http://localhost:8080/renderer/using-layers/renderer.html
    • CasparCG command: 
      PLAY 1-10 [html] http://localhost:8080/renderer/using-layers/renderer.html
  3. Connect your controller (or use the included Simple Controller )
